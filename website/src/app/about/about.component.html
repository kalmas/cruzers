What is this?

This is a website to help you relive the magic of Cruzers Seamans Club. Cruzers was a dive karaoke bar, it used to be here, but now it's gone. This site was originally built to ease navigation of the notoriously disorganized Cruzers songbook, now it serves no practical purpose... unless you wanted to throw a Cruzers reunion party... you should certainly do that.

Song data was pulled from these scans. Some of the data is still mangled from the import, but it wouldn't feel like Cruzers if it was too easy to search, you're welcome.

This site is dedicated to the memory Norfolk's best bar. The building is something new now, but to some it will be Cruzers forever.

Website is by Kyle (kalmas.net). Source available at https://github.com/kalmas/cruzers.
